## ğŸš€ Installation & Setup

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A local HTTP server (e.g., Python's `http.server`, Node.js `http-server`, or Apache)

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/jay-maker113/joptify-spotify-clone.git
   cd joptify-spotify-clone
   ```

2. **Set up the local server**
   - Using Python (recommended):
     ```bash
     python -m http.server 8000
     ```
   - Using Node.js (if you have http-server installed):
     ```bash
     npx http-server -p 8000
     ```

3. **Open in browser**
   - Navigate to `http://localhost:8000` in your web browser
   - The music player will load and display available songs
